Using the Music Player Application


This is a web page for music player application help link.
The music player application is an application
to play the music.
In this application, the user should log in or
create user to use the music player application.

The user can click 'Open' to add a song to the play-list.
The user can click 'Save' to save the song in the play-list to the database.
(The next time the user log in, it will show up in the play-list)
The user can click 'Log Out' to log out from the music player form.
(When user log out, the user should log in again to user the music player)
The user can search the song, if the title is found then it will highlight the title in the index.
If not found, it will outputting the the title that contains the search words
The user can click the song in the list box and 'Delete' to remove the song in the play-list

The play button is to play the current song.
The first button is to play the very first button.
The prev button is to play the previous song from current
The next button is to play the next song from current
The last button is to play the last song of the play-list

The user can double click the text box to clear the
list box and display the play-list and text box
